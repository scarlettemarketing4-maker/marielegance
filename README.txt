MARI ELEGANCE - Sitio demo listo para publicar
Contenido:
 - index.html  (home + product listing)
 - styles.css  (estilos)
 - products.json (productos de ejemplo)
 - checkout.html, confirmation.html (flujo de pago demo)
 - imagenes (archivos placeholders; reemplaza con fotos reales)
 - instagram-cover.png (mockup para redes)
 - halo-logo.png (logo placeholder)

Instrucciones rápidas para publicar:
1) Reemplaza los archivos de imagen en la carpeta con tus fotos reales (mismo nombre de archivo).
2) Edita 'products.json' si deseas cambiar precios o textos.
3) Cambia los enlaces de WhatsApp (https://wa.me/10000000000) por tu número internacional real.
4) Subir carpeta entera a Netlify (drag & drop) o Replit (nuevo Repl -> static files) o GitHub Pages.
5) Para aceptar pagos reales, conecta Stripe / PayPal / Mercado Pago. La demo incluye un botón "pagar con tarjeta (demo)".
